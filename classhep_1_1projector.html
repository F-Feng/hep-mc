◆ add()

template<typename T >
void hep::projector< T >::add ( std::size_t  index,
T  projection,
T  value 
)
inline

Projects a point for the distribution with the specified index to the x-axis at the value projection and sets the value to value. The weight of the point is automatically multiplied with value. Note that if you call this function multiple times with same values of index and projection, the result is not same as calling this function with the sum of the values a single time. The difference is due to the standard deviation which is calculated with the square of value.
