template<typename T >
vegas_pdf<T> hep::vegas_refine_pdf ( T  alpha,
vegas_pdf< T > const &  pdf,
std::vector< T > const &  data 
)
inline

Refines the pdf using data, which must be the binned square-function values, and returns the new pdf. The process can be controlled by the parameter alpha which is documented in the Vegas publication [2] Vegas2. This function's code is derived from Thomas Hahn's refine_grid from the CUBA VEGAS implementation Cuba.
