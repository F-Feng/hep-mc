Detailed Description

template<typename T>
struct hep::mc_point< T >

A random point \( \vec{x} \in [0,1]^d \) in the \( d \)-dimensional hypercube with an associated weight.

Examples:
mpi_vegas_example.cpp, multi_channel_example.cpp, vegas_example.cpp, vegas_grid.cpp, and vegas_stop_after_precision.cpp.

Member Data Documentation

template<typename T>
std::vector<T> const& hep::mc_point< T >::point

The coordinates of this point of the hypercube. The dimension can be obtained using point.size().

template<typename T>
T hep::mc_point< T >::weight

The weight \( w \) of this point. Depending on the integration algorithm the weight might be constant over the entire hypercube (e.g. plain) or dependent on the region in which it lies (e.g. vegas).
